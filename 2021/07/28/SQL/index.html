SQL基本語法

SELECT:查詢資料;

AS: 設定欄位別名

SELECT DISTINCT: 去除重複資料只顯示一筆;

WHERE: 篩選條件

AND、OR、NOT: 連接多個條件式

ex:

BETWEEN…AND: 設定篩選範圍

ex:

IN: 指定多個篩選值

ex:

LIKE: 設定字串比對的篩選值

萬用字元的使用

  • _ 代表單個任意字元
  • % 代表零或多個任意字元

ex:查出第二個字為”中”且為三個字的姓名

ex:查出第二個字為”中”,長度不限的姓名

ex:查出號碼為0918開頭

ORDER BY: 設定查詢結果的排序

排序方式有兩種

  1. ASC:遞增排序，由小到大，也是未指定時的預設排法。
  2. DESC:遞減排序，由大到小。

ex:生日遞減排序

多欄位排序

依條件依序排列,條件與條件之間逗號分隔

ex:性別遞增排序後,遞減生日排序

LIMIT: 設定查詢顯示的筆數

可以設定查詢後由哪筆開始顯示,並顯示多少筆

SQL中是由0開始,若從0筆開始

若資料由第一筆開始, 0可直接省略如下

  • ORDER BY 搭配 LIMIT 可取得前X筆 或 後X筆 資料
